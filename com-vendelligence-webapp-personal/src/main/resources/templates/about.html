Vendelligence

Research support and filtering for enterprise software vendor products and open source projects.

About Vendelligence

Vendelligence is a curated research support and filtering tool that focuses on indexed public resources from commercial software vendors and open source projects.

It relies on simple PRE and POST processing search Filters to assist with query research. The filters cover vendor product documentation (help libraries), API documentation (code libraries), vendor community forums, engineering blogs from trusted public architects/engineers that are regarded as domain experts, knowledge providers like 3rd party media technical expert sites, and Q&A sites relevant to the technology (like stack overflow).

Vendelligence integrates with the Google Custom Search JSON API as I wanted a way to control the filters and search quality while being able to integrate it with an application that runs locally or on a remote hosted service.

This version runs in simple search-only mode. The GitHub project shows an extended version of the tool using search-and-store mode, so you can save, retrieve, update, and delete your queries and annotate them.

The private personal edition build uses a larger number of vendors and open source projects and connects to additional APIs to support my own workflows.

For additional information about the project, refer to the informazine.com projects platform Vendelligence page.
